Here are 8 points with examples on how to dynamically change the JavaScript script src using JavaScript:
Create a new script element and set the src attribute to the desired location:
let script = document.createElement('script');
script.src = '/newScript.js';
Append the script element to the DOM:
document.body.appendChild(script);
Modify the existing script element's src attribute:
let oldScript = document.querySelector('script');
oldScript.src = '/newScript.js';
 Use the replace() method on the existing script element's src attribute:
let myScript = document.querySelector('script');
myScript.src.replace('oldScript.js', 'newScript.js');
Use the setAttribute method:
let myScript = document.querySelector('script');
myScript.setAttribute('src', '/newScript.js');
 Use the insertAdjacentHTML method:
let myScript = document.querySelector('script');
myScript.insertAdjacentHTML('beforebegin', `<script src="/newScript.js"></script>`);
