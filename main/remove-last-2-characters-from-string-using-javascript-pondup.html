Use the slice() method to return a substring from the given string, beginning at the start index and up to but not including the end index. 
Pass -2 as the second argument to slice() to remove the last two characters from the string. For example: 
myString.slice(0, -2);
Use the substr() method to return a specified number of characters from a string, beginning at the start index.

Pass -2 as the second argument to substr() to remove the last two characters from the string. For example:

myString.substr(0, myString.length - 2);
Use the substring() method to return a substring from the given string, beginning at the start index and up to but not including the end index.
Pass the string length minus two as the second argument to substring() to remove the last two characters from the string. For example:
myString.substring(0, myString.length - 2);
Use the replace() method to return a modified string, replacing specified parts of the original string with a new substring.
Pass a regular expression as the first argument to replace() to remove the last two characters from the string. For example:
myString.replace(/.{2}$/, "");
